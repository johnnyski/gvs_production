monthly_2A-52: Process Rain Existence for Monthly Data Set

Last updated: 8/17/98 
Synopsis 
monthly_2A-52 [-v] [-o output_dir] [-w working_dir] 
               [-V TSDIS_prod_version#] [-p 2A-53-product] 
               [-versions_file filename] [-not_report_monitorps]
               top_input_dir tapeid program prog_options
where,
See options.
top_input_dir : Directory where the daily data sets are located under.  
                See Input Files section below.
tapeid        : Tape ID of the monthly data set.
program       : 2A-52i (Rain existence program).
prog_options  : Options for program.

Description

This program will call 2A-52i multiple times to produce a monthly rain existence or 2A-52 product for one site.

Input Files

  1. A month worth of rain rate granules in HDF format located under top_input_dir that has the following hierarchial structure:
    2.                          | daily_1 --> gvs_2A-53-{c|d|w|v4}_tapeid --> HDF file(s)
       top_input_dir --> | daily_2 --> gvs_2A-53-{c|d|w|v4}_tapeid --> HDF file(s)
                                   ...
                         | daily_N --> gvs_2A-53-{c|d|w|v4}_tapeid --> HDF file(s)
            
        HDF file(s) must follow TSDIS file naming convention and may
        be compressed. The file format is as followed: 2A53.c.n.r.v.HDF[.gz|.Z]; 
        see file's legend listed in Output File section.

Temporary File

$working_dir/re_<pid>.ASCII.

Output File

2A52.c.r.v - Ascii file containing rain existence data of one site for one month. The entries are sorted based on time of VOSs. The output filename will be constructed by this program. 
   legend:
      c = yymmdd (date of the first day in this monthly collection)
      n = granule #
      r = GV radar or site designation
      v = version number of TSDIS product

Options

monthly_2A-52 recognizes the following options:

-v: Output steps of program execution.

-o: Specify output directory. Default: './'.

-w: Specify working directory. Default: './'.

-V: Specify TSDIS' product version number. Default: 1.

-p: Specify rain rate product to be used as input. Available values: 2A-53-{c|d|w|v4}. Default: 2A-53-v4.

-versions_file: Specify filename that contains TSDIS products'  version numbers. Default = $GVS_DATA_PATH/products_version_numbers.  See the default file for format when creating this file.

-not_report_monitorps: Do not send status messages to monitorps.
 

Return value

monthly_2A-52 returns zero for successful completion; -1 for failure; and -2 for interruption.
